Neovim

Neovim is a fork of vim that provides saner defaults and other benefits, I personally believe Vim is the best text editor ever, the syntax higlighting is amazing, the editor commands save so much time, it's just easier.

In the past I used Emacs (With WebMode) as a text editor, which was nice, but when I joined the Tilde.team server I was shocked at how ugly it was, it was a depressing grey, it was impossible to see anything and for some reason the auto close tags feature didn't work so I decided to use Vim instead. Vim was an improvement, instead of depressing grey colors it was bright and colorful, it was easy to see and it was easy to use (at least for me)

I finally uninstaled Emacs, yes, it can be used to check emails but for some reason it NEVER works for me (neither does fetchmail for some reason) and it also has calendars and games but to me that's a deficiency, it's way too bloated. Many people
